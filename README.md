# FinScholar

___

## What is it?

FinScholar is an app made by students for students. 
It's a great way to manage savings, set a budget, and track your expenses.
It's free to use and easy to get started.

## How to use it?

Simply sign up and start using the app. Once on the app, you can view an overview of your balance, recent transactions, savings goal, and current monthly budget in the home page.
